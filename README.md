# CP PWA App

Vue 3 + Vuetify 3 + PWA로 구축된 설치형 웹앱입니다.

## 기술 스택

- **프레임워크**: Vue 3 (Composition API)
- **UI**: Vuetify 3
- **번들러**: Vite
- **라우팅**: Vue Router 4
- **PWA**: vite-plugin-pwa
- **디버깅**: vite-plugin-vue-devtools

## 주요 기능

- 🏠 홈페이지
- ⚙️ 설정 페이지
- 📱 PWA 설치 지원
- 🔄 오프라인 캐싱
- 🚀 GitHub Pages 배포

## 개발 서버 실행

```bash
npm run dev
```

## 빌드

```bash
npm run build
```

## 배포

GitHub Actions를 통해 자동으로 GitHub Pages에 배포됩니다.
